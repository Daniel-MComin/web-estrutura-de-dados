Pilha (Stack)

Estrutura de dados LIFO (Last In, First Out) - O último a entrar é o primeiro a sair

O que é uma Pilha?

Uma pilha é uma estrutura de dados que armazena elementos de forma sequencial que segue o princípio LIFO (Last In, First Out), ou seja, o último elemento inserido na pilha é o primeiro a ser removido, como uma pilha de pratos ou uma pilha de livros. Sempre que um novo elemento é inserido (ou empilhado) damos a ele o nome de “topo”,

Operações principais:

  • • Push: O(1) - Adiciona elemento no topo
  • • Pop: O(1) - Remove elemento do topo
  • • Peek: O(1) - Visualiza o topo sem remover
  • • isEmpty: O(1) - Verifica se está vazia

Algumas aplicações são:

  • • Desfazer e refazer em Aplicações, os famosos “Undo” e “Redo”. (O famoso CRTL-Z e seu irmão menos famoso CRTL-Y).
  • • Navegação em “browsers, com o botão “Voltar”.
  • • Torre de Hanoi (apenas o disco no topo pode ser movido).
  • • Problemas recursivos em matemática, como cálculo de fatoriais ou números de Fibonacci, dependem de pilhas nos bastidores.
  • • Na execução de programas, cada chamada de função é armazenada em uma pilha. A função mais recente chamada fica no topo e deve ser concluída antes que as anteriores possam continuar.

Exemplo de Código


class Queue {
  constructor() {
    this.items = [];
  }

  enqueue(element) {
    this.items.push(element);
  }

  dequeue() {
    if (this.isEmpty()) return null;
    return this.items.shift();
  }

  front() {
    if (this.isEmpty()) return null;
    return this.items[0];
  }

  isEmpty() {
    return this.items.length === 0;
  }
}
