Busca Linear (Linear Search)

Algoritmo de busca que percorre o array sequencialmente comparando cada elemento com o alvo

Características:

  • Complexidade: O(n) no pior caso
  • Vantagem: Funciona em arrays não ordenados
  • Desvantagem: Lento para arrays grandes
  • Uso ideal: Arrays pequenos ou não ordenados
  • Melhor Caso: Ocorre quando o primeiro item do vetor é o item procurado O(1)

Como funciona?

  • A busca inicia a partir do primeiro item da lista.
  • O algoritmo compara o valor alvo com o valor do elemento atual.
  • Se o valor for encontrado, o algoritmo retorna a posição (índice) do elemento.
  • Caso contrário, ele se move para o próximo elemento da lista.
  • O processo é repetido até que o elemento seja encontrado ou que todos os itens da lista tenham sido verificados.
