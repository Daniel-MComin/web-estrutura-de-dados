Fila (Queue)

Estrutura de dados FIFO (First In, First Out) - O primeiro a entrar é o primeiro a sair

Como funciona?

A fila funciona como uma fila de banco: o primeiro cliente a chegar é o primeiro a ser atendido. Novos elementos são adicionados no final da fila e removidos do início.

Operações principais:

  • enqueue(): Adiciona um elemento no final da fila - O(1)
  • dequeue(): Remove e retorna o elemento do início - O(n) em JS, O(1) em listas ligadas
  • front(): Retorna o elemento do início sem removê-lo - O(1)
  • isEmpty(): Verifica se a fila está vazia - O(1)

Exemplo de Código


class Queue {
  constructor() {
    this.items = [];
  }

  enqueue(element) {
    this.items.push(element);
  }

  dequeue() {
    if (this.isEmpty()) return null;
    return this.items.shift();
  }

  front() {
    if (this.isEmpty()) return null;
    return this.items[0];
  }

  isEmpty() {
    return this.items.length === 0;
  }
}
