Hash Encadeada

Tabela hash com tratamento de colisões por encadeamento utilizando listas encadeadas.

Características:

  • Objetivo: Armazenar dados de forma eficiente, permitindo operações de inserção, busca e remoção em tempo médio O(1) (tempo constante).
  • Como ocorre colisões? Ocorre quando a Função Hash calcula o mesmo índice (slot) na tabela para duas chaves diferentes.
  • Como ele soluciona a colisão? Em vez de armazenar o dado diretamente no slot, cada posição da tabela (array) armazena o cabeçalho (início) de uma Lista Encadeada, onde o elemento vai ser adicionado nessa lista encadeada.

Complexidade:

  • Inserir: O(1)
  • Buscar: O(1)
  • Remover: O(1)
  • Pior caso: O(n) se todas as chaves forem para um único slot.
  • Vantagens: Simples de implementar e evita que a tabela "encha", pois novos elementos podem sempre ser adicionados à lista.
  • Desvantagens: Se a função hash for ruim, as listas podem ficar muito longas, degradando o tempo de busca para $O(n)$ (tempo linear no pior caso).

Exemplo de Código


  typedef struct Node {
  int data;
  struct Node* next;
} Node;

#define TABLE_SIZE 10
Node* hashTable[TABLE_SIZE];

int hashFunction(int key) {
    return key % TABLE_SIZE;
}

void insert(int value) {
    int index = hashFunction(value);

    Node* newNode = (Node*)malloc(sizeof(Node));
    if (newNode == NULL) {
        printf("Erro de alocação de memória.\n");
        return;
    }
    newNode->data = value;

    newNode->next = hashTable[index];
    hashTable[index] = newNode;
    
    printf("Inserido: %d no índice %d\n", value, index);
}

int search(int value) {
    int index = hashFunction(value);

    Node* current = hashTable[index];
    
    while (current != NULL) {
        if (current->data == value) {
            return 1;
        }
        current = current->next;
    }
    
    return 0;
}

void display() {
    printf("\n--- Conteúdo da Tabela Hash (Encadeamento) ---\n");
    for (int i = 0; i < TABLE_SIZE; i++) {
        printf("Índice %d: ", i);
        Node* current = hashTable[i];
        while (current != NULL) {
            printf("%d -> ", current->data);
            current = current->next;
        }
        printf("NULL\n");
    }
}

int main() {
    for (int i = 0; i < TABLE_SIZE; i++) {
        hashTable[i] = NULL;
    }

    insert(15);
    insert(25);
    insert(7);
    insert(100);
    insert(10);

    display();

    if (search(25)) {
        printf("Busca: 25 ENCONTRADO\n");
    } else {
        printf("Busca: 25 NÃO ENCONTRADO\n");
    }

    if (search(99)) {
        printf("Busca: 99 ENCONTRADO\n");
    } else {
        printf("Busca: 99 NÃO ENCONTRADO\n");
    }

    return 0;
}
