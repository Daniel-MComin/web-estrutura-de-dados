Skip List

Estrutura de dados probabilística que permite busca rápida em O(log n)

Como funciona?

Skip List é uma estrutura de dados probabilística baseada em listas ligadas que permite busca, inserção e remoção em tempo O(log n) em média. Funciona como múltiplas listas ligadas ordenadas em diferentes níveis, onde níveis superiores contêm subconjuntos dos elementos dos níveis inferiores.

Operações principais:

  • enqueue(): Adiciona um elemento no final da fila - O(1)
  • dequeue(): Remove e retorna o elemento do início - O(n)
  • front(): Retorna o elemento do início sem removê-lo - O(1)
  • isEmpty(): Verifica se a fila está vazia - O(1)

Exemplo de Código


class Queue {
  constructor() {
    this.items = [];
  }

  enqueue(element) {
    this.items.push(element);
  }

  dequeue() {
    if (this.isEmpty()) return null;
    return this.items.shift();
  }

  front() {
    if (this.isEmpty()) return null;
    return this.items[0];
  }

  isEmpty() {
    return this.items.length === 0;
  }
}

Casos de Uso

  • • Implementação de bancos de dados (Redis, LevelDB)
  • • Estruturas de índice em sistemas de arquivos
  • • Alternativa simples a árvores balanceadas
  • • Tabelas de símbolos e dicionários ordenados
